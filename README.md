# Premium React Portfolio

Luxury personal portfolio web application with animated public pages and a hidden protected admin UI.

## Run locally

```bash
npm install
npm run dev
```

Create `.env.local` from `.env.example` to connect live services:

```bash
VITE_GITHUB_USERNAME=netramehandiratta89-dev
VITE_VERCEL_PROJECT_NAME=portfolio-webapp
VITE_VERCEL_DEPLOYMENT_URL=https://your-project.vercel.app
```

Public site: `/`

Hidden admin login: `/admin/login`

Demo admin credentials:

- Email: `admin@portfolio.dev`
- Password: `premium123`

## Tech

React, Vite, Tailwind CSS, Framer Motion, GSAP, React Router DOM, component-based architecture.
